# My Portfolio Website

A simple personal portfolio website built with HTML, CSS, and JavaScript.

## Files

- `index.html` — the main portfolio page
- `styles.css` — responsive visual styling
- `script.js` — navigation toggle and demo contact form behavior

## Customize

1. Replace `Your Name` with your real name in `index.html`.
2. Update the about text, projects, skills, and contact details.
3. Change links and add your own project case studies.
4. For production, host this folder on GitHub Pages, Netlify, Vercel, or any static host.

## Local preview

Open `index.html` in your browser, or use a local server such as:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.
